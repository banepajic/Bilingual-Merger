Bilingual .docx Merger

Merge two Word files (same structure) into a single bilingual table with {mlang} blocks. Based on your Tkinter script — but runs in the browser.

Used as {mlang <code>}
Ready.
Assumptions: All tables from both .docx files are processed. Questions are extracted from the second column of each table. A new question is detected automatically based on its structure (typically starts with an ID). Options begin with A., B., etc. Rows starting with ANSWER are copied from Language 1. Each table is preserved separately in the output document.
